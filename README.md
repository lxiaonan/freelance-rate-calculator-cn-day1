# 自由职业报价换算器 | Freelance Rate Calculator

## 中文介绍

### 这是什么

这是一个面向自由职业者、顾问、独立开发者和兼职从业者的实用报价网站。你只需要输入目标月收入、固定成本、可计费工时和风险缓冲，它就能快速给出更稳妥的时薪、日薪和项目报价建议。

### 核心功能

- 根据目标月收入和固定成本，换算合理的月营收目标
- 根据可计费工时、休假周数和非计费时间，估算真实出售工时
- 自动生成建议时薪、日薪和项目报价
- 内置中英双语界面切换
- 纯静态网站，可直接托管到 GitHub Pages

### 使用方式

1. 输入你的目标月收入和每月固定成本。
2. 填写每周可计费小时、休假周数、非计费时间占比和风险缓冲。
3. 如果你要单独报价某个项目，再输入预估项目工时和项目风险系数。
4. 页面会实时更新推荐的时薪、日薪和项目报价。

## English

### What It Is

This is a practical pricing website for freelancers, consultants, indie makers, and part-time professionals. Enter your income goal, fixed costs, billable time, and risk buffer to get safer hourly, daily, and project rate suggestions.

### Core Features

- Turns your target income and fixed costs into a realistic monthly revenue target
- Estimates real sellable hours from billable time, vacation weeks, and overhead
- Generates suggested hourly, daily, and project quotes instantly
- Includes a Chinese and English interface toggle
- Fully static and easy to host on GitHub Pages

### How To Use

1. Enter your target monthly income and fixed monthly costs.
2. Fill in your weekly billable hours, vacation weeks, overhead ratio, and buffer.
3. Add project hours and project risk if you want a project-specific quote.
4. Review the live pricing suggestions and adjust your inputs as needed.
